How to Run:
Compile all files in the src folder using javac filename.java
Run Main.java using java Main

Commands: 
Login: login
Sign Up: signup
Diary: diary
To-Do: todo
Make File: make file
Display File: display file
Remove File: remove file
Add a task: add task
Complete a task: complete task
Exit: exit

When writing to diary, you must make the file and enter an existing filename and you will then have the choice to overwrite or append to the file of choice.

If you wish to go back to the main menu, type anything other than the above commands and you will be sent back.

Very first time you run this program on your device, you must sign up, you cannot login.
